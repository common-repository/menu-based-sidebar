== Description ==

A smart way to display child menu items in widget sidebar. Menu based sidebar plugin has provide an option to display the child menu items in sidebar on based on selected current parent menu item.

= Features = 

 * Shortcode [mbs_menu menu_id="MENU_ID_GOES_HERE"]
 * Display menu items in sidebar on based current main menu item
 * Auto display child pages on based of current main menu item

== Frequently Asked Questions ==

#Is this allowing to display child menu items in sidebar ?

 * Yes, on basis of current main menu item, it will automatically display their all child menu items in sidebar 
 
#Can I add sidebar under page content ?   

 * Yes, You can use shortcode any where of the website where you want to display menu based listing 
 
#Can I show my footer menu items in sidebar?

 * Yes,Using the shortcode you can display listing of any menu, just need to add the menu id in shortcode [mbs_menu menu_id="MENU_ID_GOES_HERE"]
